Attendees

We’re excited to have you attend our workshop! After you signing up as soon as we set up the Event page, please read the following:

What you should bring

  • a laptop
  • your laptop’s charger
  • your enthusiasm and open mind

Should you not be able to bring a laptop and attend a Berlin event, please feel free to contact us at Twitter @cssclasses, talk to us directly or leave a comment on the current Meetup page.

Tools to download

Please install the following software on your laptop before the workshop. If you run into trouble, we’ll help you with the installation at the workshop.

Browser

You can generally use any browser you're familiar with. However, due to having the most recent CSS features and good debugging support, developers often tend to use one of the following browsers:

  • Firefox
  • Chrome

Text editor

  • Atom
  • VS Code
  • Sublime Text 3
